# ned

# Prerequisites

- Flutter SDK (supports SDK ^3.6.1)
- Dart SDK
- IDE (VS Code, Android Studio, or IntelliJ)

## Installation

```bash
# Clone the repository
git clone [repository-url]
cd ned

# Install dependencies
flutter pub get

# Run the application
flutter run -d chrome  # For web
# or
flutter run            # For mobile/desktop
```

## Dependencies

```
dependencies:
  http: ^1.2.0
  provider: ^6.1.1
  intl: ^0.19.0
```
